## ğŸš€ USAGE EXAMPLES

### 1. CLI Usage
```bash
# Basic conversion
rsj-ffmpeg -i input.mp4 -o output.mp4

# AI upscale with watermark
rsj-ffmpeg -i video.mp4 --ai-upscale 4x --watermark "RSJ" -o out.mp4

# Batch processing
rsj-ffmpeg --batch ./videos/ --enhance --export ./output/
```

### 2. Python API
```python
from rsj_ffmpeg import RSJToolkit
from rsj_ffmpeg.streaming import StreamManager
from rsj_ffmpeg.ai_engine import AIEngine

toolkit = RSJToolkit()

# Batch convert
toolkit.batch_convert(
    input_dir="./videos/",
    output_dir="./output/",
    ai_upscale="4x",
    enhance=True
)

# AI enhancement
ai_engine = AIEngine(toolkit.config)
ai_engine.upscale_video("input.mp4", "output.mp4", scale=4)

# Streaming
stream_manager = StreamManager(toolkit.config)
stream_manager.start_stream(
    stream_id="live_001",
    platform="youtube",
    key="YOUR_KEY"
)
```

### 3. REST API
```bash
# Start API server
python3 api_server.py

# Convert video
curl -X POST http://localhost:8080/api/v1/convert \
  -H "Content-Type: application/json" \
  -d '{"input_file": "input.mp4", "output_file": "output.mp4"}'

# Start stream
curl -X POST http://localhost:8080/api/v1/stream/start \
  -H "Content-Type: application/json" \
  -d '{"platform": "youtube", "key": "YOUR_KEY"}'
```
