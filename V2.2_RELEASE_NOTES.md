    output="highlights.mp4"
)

**450+ lines | 20 features**

- Beat-synchronized editing

- Style-specific color grading
- Music integration
- Template-based montages

**Quick Example:**
```python

    input_dir="./clips/",
    style="cinematic",
    duration=120,
    music="soundtrack.mp3",
    output="montage.mp4"
)
```
