RSJ-FFMPEG Documentation

Welcome to the complete documentation for RSJ-FFMPEG Power Toolkit v2.0.0 - the ultimate AI-powered multimedia automation framework.

Installation

Choose your platform and follow the installation instructions:

Linux / Mac / Termux

curl -sL https://raw.githubusercontent.com/RAJSARASWATI-JATAV/rsj-ffmpeg-toolkit/main/install.sh | bash

Windows PowerShell

iwr -useb https://raw.githubusercontent.com/RAJSARASWATI-JATAV/rsj-ffmpeg-toolkit/main/install.ps1 | iex

Manual Installation

git clone https://github.com/RAJSARASWATI-JATAV/rsj-ffmpeg-toolkit.git
cd rsj-ffmpeg-toolkit
pip install -r requirements.txt
python setup.py install

CLI Reference

The command-line interface provides powerful tools for video and audio processing.

Basic Usage

# Convert video
rsj-ffmpeg -i input.mp4 -o output.mp4

# Get video information
rsj-ffmpeg -i video.mp4

# System check
rsj-ffmpeg --system-check

# Show version
rsj-ffmpeg --version

Video Processing

# AI upscaling
rsj-ffmpeg -i input.mp4 --ai-upscale 4x -o output.mp4

# Add watermark
rsj-ffmpeg -i input.mp4 --watermark "RAJSARASWATI JATAV" -o output.mp4

# Create GIF
rsj-ffmpeg -i input.mp4 --to-gif -o output.gif

# Extract frames
rsj-ffmpeg -i input.mp4 --extract-frames ./frames/

# Video enhancement
rsj-ffmpeg -i input.mp4 --enhance -o output.mp4

Batch Processing

# Batch convert directory
rsj-ffmpeg --batch ./videos/ --format mp4 --export ./output/

# Batch with AI upscaling
rsj-ffmpeg --batch ./videos/ --ai-upscale 2x --export ./output/

# Batch with watermark
rsj-ffmpeg --batch ./videos/ --watermark "RSJ" --export ./output/

Python API

Use RSJ-FFMPEG programmatically in your Python applications.

Basic Usage

from rsj_ffmpeg import RSJToolkit

# Initialize toolkit
toolkit = RSJToolkit()

# Convert video
toolkit.convert_video(
    input_file="input.mp4",
    output_file="output.mp4"
)

# Get video info
info = toolkit.get_video_info("video.mp4")
print(info)

Batch Processing

toolkit.batch_convert(
    input_dir="./videos/",
    output_dir="./output/",
    ai_upscale="4x",
    enhance=True,
    watermark="RAJSARASWATI JATAV"
)

REST API

Control RSJ-FFMPEG remotely via REST API.

Starting the API Server

python api_server.py

The API will be available at http://localhost:8080

API Endpoints

POST /api/v1/convert

Convert a video file to a different format.

Parameter Type Description
input_file string Path to input video file
output_file string Path to output video file
format string Output format (optional, default: mp4)
POST /api/v1/enhance

Enhance a video using AI.

Parameter Type Description
input_file string Path to input video file
output_file string Path to output video file
ai_upscale string Upscale factor (2x, 4x, 8x)
POST /api/v1/stream/start

Start a live stream.

Parameter Type Description
platform string Streaming platform (youtube, twitch, facebook)
key string Stream key
input_source string Input source (webcam, file, etc.)

Streaming

Stream to multiple platforms simultaneously with real-time overlays.

Python API

from rsj_ffmpeg.streaming import StreamManager

stream_manager = StreamManager(config)

# Start YouTube stream
stream_manager.start_stream(
    stream_id="live_001",
    platform="youtube",
    key="YOUR_STREAM_KEY",
    input_source="webcam",
    overlay="logo.png",
    stats_overlay=True
)

AI Engine

Advanced AI-powered video and audio processing.

from rsj_ffmpeg.ai_engine import AIEngine

ai_engine = AIEngine(config)

# AI upscaling
ai_engine.upscale_video(
    input_file="input.mp4",
    output_file="output.mp4",
    scale=4
)

# Scene detection
scenes = ai_engine.scene_detection(
    input_file="video.mp4",
    threshold=0.4
)

# Audio separation
ai_engine.audio_separation(
    input_file="song.mp3",
    output_dir="./stems/"
)
