‚ùå FFmpeg not found error

+
Error: "FFmpeg not found in system PATH"
‚úÖ Solution:
  1. Install FFmpeg:
    • Windows: Download from ffmpeg.org and add to PATH
    • Linux: sudo apt install ffmpeg
    • macOS: brew install ffmpeg
    • Termux: pkg install ffmpeg
  2. Verify installation: ffmpeg -version
  3. Restart your terminal/IDE

‚ùå Python module import error

+
Error: "ModuleNotFoundError: No module named 'rsj_ffmpeg'"
‚úÖ Solution:
  1. Ensure you're in the correct directory
  2. Install dependencies: pip install -r requirements.txt
  3. Add to Python path: export PYTHONPATH="${PYTHONPATH}:$(pwd)"
  4. Or install as package: pip install -e .

‚ùå Permission denied errors

+
Error: "Permission denied" when running scripts
‚úÖ Solution:
  1. Make scripts executable: chmod +x install.sh
  2. Run with proper permissions: sudo ./install.sh
  3. Check file ownership: ls -la

üé¨ Video Processing Issues

‚ùå Video encoding fails

+
Error: "Encoding failed" or "Invalid codec"
‚úÖ Solution:
  1. Check FFmpeg codec support: ffmpeg -codecs
  2. Try different codec: Use h264 instead of h265
  3. Update FFmpeg to latest version
  4. Check input file integrity: ffmpeg -i input.mp4

‚ùå GPU encoding not working

+
Error: "GPU encoder not available"
‚úÖ Solution:
  1. Check GPU detection: 
    from rsj_ffmpeg.gpu_acceleration import GPUAccelerationEngine
gpu = GPUAccelerationEngine({})
print(gpu.get_gpu_info())
  2. Install GPU drivers (NVIDIA/AMD/Intel)
  3. For NVIDIA: Install CUDA toolkit
  4. Rebuild FFmpeg with GPU support

‚ùå Poor output quality

+
Issue: Output video has poor quality or artifacts
‚úÖ Solution:
  1. Increase quality setting: quality="high" or lower CRF value
  2. Use better codec: h265 instead of h264
  3. Increase bitrate: bitrate="5M"
  4. Use slower preset for better quality: preset="slow"

‚ùå Processing too slow

+
Issue: Video processing takes too long
‚úÖ Solution:
  1. Enable GPU acceleration
  2. Use faster preset: preset="fast"
  3. Lower output resolution
  4. Enable caching to avoid reprocessing
  5. Use distributed processing for large batches

üîä Audio Issues

‚ùå No audio in output

+
Error: Output video has no audio
‚úÖ Solution:
  1. Check if input has audio: ffmpeg -i input.mp4
  2. Ensure audio codec is supported
  3. Use -c:a copy to copy audio stream
  4. Check audio mapping in FFmpeg command

‚ùå Audio sync issues

+
Issue: Audio and video out of sync
‚úÖ Solution:
  1. Use -async 1 flag in FFmpeg
  2. Re-encode both audio and video
  3. Check frame rate consistency
  4. Use -vsync cfr for constant frame rate

üíæ Performance & Memory

‚ùå Out of memory errors

+
Error: "MemoryError" or system runs out of RAM
‚úÖ Solution:
  1. Process videos in smaller chunks
  2. Lower output resolution
  3. Close other applications
  4. Use streaming processing instead of loading entire file
  5. Enable swap space on Linux

‚ùå Cache filling up disk

+
Issue: Cache directory consuming too much disk space
‚úÖ Solution: 
from rsj_ffmpeg.cache import CacheManager

cache = CacheManager({})

# Clean old cache
cache.clean_cache(max_age_days=7)

# Or clear all
cache.clear_all_cache()

# Check stats
stats = cache.get_cache_stats()
print(f"Cache size: {stats['total_size_mb']:.2f} MB")
