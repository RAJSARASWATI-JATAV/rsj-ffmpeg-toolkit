## ü§ñ GPT Director

Edit videos using natural language commands powered by GPT-4.

### Quick Start

```python
from rsj_ffmpeg import GPTDirector

director = GPTDirector(
    config={},
    api_key="your-openai-api-key"
)

# Edit with natural language
result = director.edit_from_prompt(
    input_files=["clip1.mp4", "clip2.mp4"],
    prompt="Create a 2-minute energetic highlight reel",
    output="highlight.mp4",
    duration=120
)
```

### Features

- **Natural Language Commands** - Describe what you want in plain English
- **Style Templates** - Cinematic, vlog, commercial, etc.
- **Script-Based Editing** - Edit from screenplay format
- **Content Analysis** - Automatic scene detection
- **Smart Clip Selection** - AI chooses best moments

### Examples

```python
# Cinematic edit
director.edit_from_prompt(
    input_files=["footage.mp4"],
    prompt="Create cinematic video with slow motion and dramatic music",
    style="cinematic",
    music="soundtrack.mp3",
    output="cinematic.mp4"
)

# Script-based
director.edit_from_script(
    script="screenplay.txt",
    footage_dir="./footage/",
    output="movie.mp4"
)

# Content analysis
analysis = director.analyze_content(
    video_file="video.mp4",
    analyze_audio=True,
    detect_scenes=True
)
```

## üé¨ Montage Engine

Create professional montages automatically with beat synchronization.

### Quick Start

```python
from rsj_ffmpeg import MontageEngine

engine = MontageEngine(config={})

# Auto-create montage
result = engine.create_auto_montage(
    input_dir="./clips/",
    output="montage.mp4",
    style="cinematic",
    duration=120,
    music="music.mp3"
)
```

### Available Styles

- **cinematic** - Slow, dramatic editing
- **sports** - Fast-paced, energetic
- **travel** - Medium pace, scenic
- **wedding** - Romantic, elegant
- **vlog** - Casual, personal
- **action** - Very fast, intense
- **music_video** - Beat-synchronized

### Beat-Synced Editing

```python
# Sync cuts to music beats
result = engine.create_beat_synced_montage(
    input_dir="./clips/",
    music="song.mp3",
    output="music_video.mp4",
    style="music_video"
)
```

## ‚òÅÔ∏è Cloud Processor

Process videos in the cloud with multi-provider support.

### Supported Providers

- **AWS** - MediaConvert + S3
- **Google Cloud** - Transcoder + Storage
- **Azure** - Media Services + Blob
- **Cloudflare** - Stream

### Quick Start

```python
from rsj_ffmpeg import CloudProcessor

# Initialize AWS
cloud = CloudProcessor(
    provider="aws",
    credentials={
        "access_key": "YOUR_KEY",
        "secret_key": "YOUR_SECRET",
        "region": "us-east-1"
    }
)

# Process video
result = cloud.process_video(
    input_file="large_video.mp4",
    operations=[
        {"type": "upscale", "factor": 4},
        {"type": "compress", "bitrate": "5M"}
    ],
    output_bucket="my-bucket",
    output_key="processed/video.mp4"
)

# Monitor progress
status = cloud.get_job_status(result['job_id'])

# Download result
cloud.download_result(result['job_id'], "output.mp4")
```

### Batch Processing

```python
# Process multiple videos
videos = ["v1.mp4", "v2.mp4", "v3.mp4"]

for video in videos:
    cloud.process_video(
        input_file=video,
        operations=[{"type": "compress"}],
        output_bucket="batch-output"
    )
```

## üîç Content Analyzer

AI-powered video content analysis.

### Quick Start

```python
from rsj_ffmpeg import ContentAnalyzer

analyzer = ContentAnalyzer(config={})

# Comprehensive analysis
analysis = analyzer.analyze_video(
    video_file="video.mp4",
    analyze_scenes=True,
    analyze_audio=True,
    analyze_quality=True,
    analyze_content=True
)

print(f"Duration: {analysis['metadata']['duration']}s")
print(f"Scenes: {len(analysis['scenes'])}")
print(f"Quality Score: {analysis['quality']['overall_score']}")
```

### Features

- **Scene Detection** - Automatic scene change detection
- **Audio Analysis** - Speech/music detection, loudness
- **Quality Assessment** - Resolution, bitrate, FPS scoring
- **Content Classification** - Video type detection
- **Improvement Suggestions** - AI recommendations

### Classification

```python
# Classify video type
classification = analyzer.classify_video("video.mp4")

print(f"Type: {classification['type']}")
print(f"Confidence: {classification['confidence']}")
print(f"Recommendations: {classification['recommendations']}")
```

### Suggestions

```python
# Get improvement suggestions
suggestions = analyzer.suggest_improvements("video.mp4")

for suggestion in suggestions:
    print(f"{suggestion['priority']}: {suggestion['suggestion']}")
    print(f"Action: {suggestion['action']}")
```

## üÜò Troubleshooting

### GPT Director Issues

**Problem:** "OpenAI not installed"  
**Solution:** `pip install openai`

**Problem:** "API key not found"  
**Solution:** Set `OPENAI_API_KEY` environment variable

### Cloud Processing Issues

**Problem:** "Credentials not found"  
**Solution:** Configure credentials in config.json or environment variables

**Problem:** "Upload failed"  
**Solution:** Check bucket permissions and network connectivity

### Analytics Dashboard Issues

**Problem:** "Port already in use"  
**Solution:** Change port in `analytics_v2.py` or kill existing process
